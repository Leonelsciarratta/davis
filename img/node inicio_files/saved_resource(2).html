                                                                              Kevin Garcia Agustina García • 4 years ago

                                                                              Yo también he llegado ha ese apartado y me salia exactamente lo mismo.
                                                                              Eso es debido a que cuando en el fichero server.js hace la llamada a route(handle, pathname), en el fichero route.js solo se le pasa por parámetros el "pathname". Nos indica que estamos intentando pasarle dos objetos.

                                                                              En esa parte del libro faltaría una captura de pantalla o el código de route.js que tendría que ser el siguiente:
                                                                              _____________________________________
                                                                              function route(handle, pathname) {
                                                                              console.log("A punto de rutear una peticion para " + pathname);
                                                                              if (typeof handle[pathname] === 'function') {
                                                                              return handle[pathname]();
                                                                              } else {
                                                                              console.log("No se ha encontrado manipulador para " + pathname);
                                                                              }
                                                                              }

                                                                              exports.route = route;
                                                                              _____________________________________

                                                                              Miraríamos si existe en ese apartado del array una función y si existe, haríamos un return de ella (devolviendo así el console.log que hay).

                                                                              Mas adelante, también faltarían algunas partes del código cuando se explica la función "exec()".
