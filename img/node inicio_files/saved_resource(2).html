                                                                              Eso es debido a que cuando en el fichero server.js hace la llamada a route(handle, pathname), en el fichero route.js solo se le pasa por parmetros el "pathname". Nos indica que estamos intentando pasarle dos objetos.

                                                                              En esa parte del libro faltara una captura de pantalla o el cdigo de route.js que tendra que ser el siguiente:
                                                                              _____________________________________
                                                                              function route(handle, pathname) {
                                                                              console.log("A punto de rutear una peticion para " + pathname);
                                                                              if (typeof handle[pathname] === 'function') {
                                                                              return handle[pathname]();
                                                                              } else {
                                                                              console.log("No se ha encontrado manipulador para " + pathname);
                                                                              }
                                                                              }

                                                                              exports.route = route;
                                                                              _____________________________________

                                                                              Miraramos si existe en ese apartado del array una funcin y si existe, haramos un return de ella (devolviendo as el console.log que hay).
